Instalasi Jurpopage :

- Buatlah database MySQL baru, (misal: namai dengan "jurpopage")

- melalui phpmyadmin, aktiflah ke database "jurpopage" anda, kemudian import file script database MySQL "jurpopage.sql"

- buka dan ubah file library/configuration.php
	$db_name = "jurpopage"; /* change it to your database name */
	$db_user = "root"; /* change it to your database user name */
	$db_passwd = "root"; /* change it to your database password */
  
- Kemudian silahkan login ke website baru anda ini, 
  LOGIN ID : admin
  PASSWORD : 1
  Catatan, jangan lupa untuk merubah password default diatas sesuai keinginan anda.
