Jurpopage Installation :

- Create new MYSQL database, (example: name it "jurpopage")

- Login to your phpmyadmin, active your "jurpopage" database, then import MySQL database script file "jurpopage.sql"

- Open file library/configuration.php
	$db_name = "jurpopage"; /* change it to your database name */
	$db_user = "root"; /* change it to your database user name */
	$db_passwd = "root"; /* change it to your database password */
  
- Then, try login at your new page, 
  LOGIN ID : admin
  PASSWORD : 1
  note, don't forget to change this default password
